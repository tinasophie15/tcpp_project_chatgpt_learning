# ChatGPT and Student Learning

## Project Overview

This project is part of the course
**“Topics in Cognitive Psychology: Psychological Interventions Using Digital Technologies”**
at the University of Graz.

The goal of this website is to present and critically evaluate a meta-analysis on the impact of ChatGPT and generative AI on student learning called **Does ChatGPT enhance student learning? A systematic review and 
meta-analysis of experimental studies** by *Deng et al.* from 2024.

The website includes:

* a structured overview of the meta-analysis,
* a discussion of key findings,
* an evaluation of the study quality,
* implications for relevant stakeholders and
* an individual study example by each contributor.

---

## Research Focus

This project explores the question:

> Does ChatGPT enhance student learning?

Based on current research, ChatGPT can support:

* academic performance,
* motivation and
* higher-order thinking.

However, its effectiveness strongly depends on:

* How it is used.
* The user’s experience.
* The learning context.
